Evaluation Protocol & Statistical Controls

Institutional Methodology & Governance Framework

1. Forecast Structure

DDL-69 produces calibrated probability distributions, multi-horizon outcome forecasts, and ensemble-weighted tool contributions. No deterministic trade signals are generated.

2. Ensemble Construction

  • Multi-tool ensemble weighting
  • Learned convex combination weights
  • Non-zero priors to prevent dominance
  • Weight regularization constraints
Guarantee: No single model can dominate final output allocation.

3. Calibration

  • Reliability diagrams
  • Brier score minimization
  • Isotonic regression or Platt scaling
  • Out-of-fold calibration only
Guarantee: Calibration uses holdout data exclusively. No in-sample probability smoothing.

4. Backtesting Protocol

  • Purged K-Fold Cross Validation
  • Embargo periods
  • Combinatorial Purged Cross Validation (CPCV)
  • Rolling walk-forward validation
Leak Prevention: No forward-looking features. Strict timestamp alignment. Feature generation cutoff enforcement.

5. Performance Metrics

  • Brier Score
  • Log Loss
  • Expected Calibration Error
  • Probabilistic Sharpe (simulated)
  • Monte Carlo distribution stress outcomes
Disclosure: Risk metrics shown are derived from simulated return paths based on probabilistic outputs and are not realized live trading results.

7. Governance & Oversight

  • Versioning system (DDL-69 v0.8)

  • Known limitations
DDL-69 does not execute trades autonomously. It produces probabilistic forecasts for analytical purposes.
